How to run
----------
to launch instances : java -cp app-environment-bootstrapper-client-0.0.1-jar-with-dependencies.jar -DCONFIG_FILE_PATH=/EBS/app-environment-bootstrapper-client-0.0.1-configuration/ -DMODE=LAUNCH com.cognizant.jpaas2.gateway.Main
to terminate instances : java -cp app-environment-bootstrapper-client-0.0.1-jar-with-dependencies.jar -DCONFIG_FILE_PATH=D:/app-environment-bootstrapper-client-0.0.1-configuration/ -DMODE=TERMINATE com.cognizant.jpaas2.gateway.Main
